# Инструкция по настройке SEO и аналитики

## Файлы для индексации
✅ **robots.txt** - создан в корне сайта
✅ **sitemap.xml** - создан в корне сайта

## Что нужно настроить после размещения на хостинге:

### 1. Обновить домен в файлах
Замените `https://your-domain.com` на ваш реальный домен во всех файлах:
- `index.html` (множество мест)
- `robots.txt` (строка Sitemap)
- `sitemap.xml` (все URL)

### 2. Яндекс.Метрика
1. Зарегистрируйтесь на https://metrika.yandex.ru/
2. Создайте счетчик для вашего сайта
3. Получите ID счетчика (числовой код)
4. В `index.html` замените `YOUR_YANDEX_COUNTER_ID` на ваш ID (2 места)
5. Добавьте код верификации: замените `YOUR_YANDEX_VERIFICATION_CODE`

**Возможности Яндекс.Метрики:**
- Аналитика посещений
- Карты кликов и скроллинга
- Вебвизор (запись сеансов пользователей)
- Цели и конверсии
- E-commerce треккинг

### 3. Google Analytics (GA4)
1. Создайте аккаунт на https://analytics.google.com/
2. Создайте ресурс GA4
3. Получите идентификатор измерения (формат: G-XXXXXXXXXX)
4. В `index.html` замените `G-XXXXXXXXXX` на ваш ID
5. Настройте события и конверсии в интерфейсе GA4

### 4. Google Search Console
1. Зайдите на https://search.google.com/search-console/
2. Добавьте свой сайт
3. Выберите способ верификации "HTML-тег"
4. Скопируйте код верификации
5. В `index.html` замените `YOUR_GOOGLE_VERIFICATION_CODE`
6. После верификации загрузите `sitemap.xml` в Search Console

### 5. Яндекс.Вебмастер
1. Зайдите на https://webmaster.yandex.ru/
2. Добавьте свой сайт
3. Пройдите верификацию (meta-тег)
4. В `index.html` замените `YOUR_YANDEX_VERIFICATION_CODE`
5. После верификации загрузите `sitemap.xml`

### 6. Favicon
Создайте и разместите в корне сайта:
- `favicon.ico` (16x16, 32x32)
- `apple-touch-icon.png` (180x180)

Можно использовать генераторы:
- https://realfavicongenerator.net/
- https://favicon.io/

### 7. Open Graph изображение
Создайте изображение для социальных сетей:
- Размер: 1200x630 px
- Формат: JPG или PNG
- Название: `og-image.jpg`
- Разместите в папке `images/`

### 8. Обновить контактные данные
В `index.html` в секции Schema.org замените:
- `+7-XXX-XXX-XX-XX` - телефон
- `Ваш город` - город
- `your_instagram_handle` - Instagram username

### 9. После размещения на хостинге:

**Проверьте работу:**
- https://validator.w3.org/ - валидация HTML
- https://search.google.com/test/rich-results - проверка структурированных данных
- https://cards-dev.twitter.com/validator - проверка Twitter Card
- https://developers.facebook.com/tools/debug/ - проверка Open Graph

**Отправьте sitemap.xml в поисковики:**
- Google Search Console: вкладка "Файлы Sitemap"
- Яндекс.Вебмастер: раздел "Индексирование → Файлы Sitemap"

**Настройте цели в аналитике:**
- Клик на кнопку "ОСТАВИТЬ ЗАЯВКУ"
- Отправка формы обратной связи
- Клик на проекты
- Клик на пакетные предложения
- Переход в Instagram

## Дополнительные рекомендации SEO:

### Скорость загрузки
- Оптимизируйте изображения (используйте WebP формат)
- Включите gzip сжатие на сервере
- Настройте кэширование браузера
- Используйте CDN для статических файлов

### Мобильная версия
✅ Сайт адаптивный и готов к индексации Mobile-First

### HTTPS
⚠️ Обязательно настройте SSL-сертификат (Let's Encrypt бесплатно)

### Контент
- Добавьте Alt-теги ко всем изображениям
- Используйте заголовки H1-H6 по иерархии
- Добавьте больше уникального текста на страницу
