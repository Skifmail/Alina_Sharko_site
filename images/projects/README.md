# Инструкция по добавлению фотографий к проектам

## Структура папок

Для каждого проекта создана отдельная папка:
- `project_1/` - фото для Проекта 1
- `project_2/` - фото для Проекта 2  
- `project_3/` - фото для Проекта 3
- `project_4/` - фото для Проекта 4
- `project_5/` - фото для Проекта 5
- `project_6/` - фото для Проекта 6

## Именование файлов

В каждой папке должно быть **ровно 9 фотографий** с именами:
```
1.jpeg
2.jpeg
3.jpeg
4.jpeg
5.jpeg
6.jpeg
7.jpeg
8.jpeg
9.jpeg
```

## Пример для Проекта 1

```
images/projects/project_1/
├── 1.jpeg  <- Первая фотография (будет показана первой в галерее)
├── 2.jpeg
├── 3.jpeg
├── 4.jpeg
├── 5.jpeg
├── 6.jpeg
├── 7.jpeg
├── 8.jpeg
└── 9.jpeg  <- Последняя фотография
```

## Важно!

1. **Формат файлов**: только `.jpeg` или `.jpg`
2. **Количество фото**: ровно 9 штук в каждой папке
3. **Названия**: строго `1.jpeg`, `2.jpeg`, ... `9.jpeg` (с цифрами от 1 до 9)
4. **Обложка проекта**: файлы `project_1.jpeg`, `project_2.jpeg` и т.д. в папке `/images/projects/` используются как обложки на главной странице

## Как добавить фотографии

1. Откройте папку нужного проекта (например, `project_1/`)
2. Скопируйте 9 ваших фотографий
3. Переименуйте их по порядку: `1.jpeg`, `2.jpeg`, ... `9.jpeg`
4. Обновите страницу в браузере - галерея заработает автоматически

## Обложки проектов

Файлы в корне папки `projects/`:
- `project_1.jpeg` - обложка Проекта 1 (показывается в сетке)
- `project_2.jpeg` - обложка Проекта 2
- `project_3.jpeg` - обложка Проекта 3
- `project_4.jpeg` - обложка Проекта 4
- `project_5.jpeg` - обложка Проекта 5
- `project_6.jpeg` - обложка Проекта 6

Эти файлы НЕ являются частью галереи, они только для отображения на главной странице.
