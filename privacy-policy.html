Политика в отношении обработки персональных данных

Важно: Данная страница находится в разработке. Полный текст политики конфиденциальности будет добавлен позже.

1. Общие положения

Настоящая Политика в отношении обработки персональных данных (далее — Политика) разработана в соответствии с Федеральным законом от 27.07.2006 №152-ФЗ «О персональных данных».

2. Оператор персональных данных

Индивидуальный предприниматель Шарко А.В.

3. Цели обработки персональных данных

Обработка персональных данных осуществляется в целях предоставления услуг по декорированию мероприятий, консультационных услуг и для связи с клиентами.

4. Категории персональных данных

Обрабатываются следующие категории персональных данных:

  • Номер телефона
  • Адрес электронной почты
  • Информация о планируемом мероприятии
  • Информация о бюджете мероприятия

5. Права субъекта персональных данных

Субъект персональных данных имеет право:

  • Получать информацию, касающуюся обработки его персональных данных
  • Требовать уточнения, блокирования или уничтожения персональных данных
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия или бездействие Оператора

6. Контактная информация

Для получения более подробной информации о политике конфиденциальности или для отзыва согласия на обработку персональных данных, пожалуйста, свяжитесь с нами.
